# Ban-Chang-Go
재난 상황에서 의료 취약계층 지원 시스템  test


## 프로젝트 소개
재난 상황에서 의료 취약계층을 지원하기 위해 AI를 활용한 시스템입니다. 
이 시스템은 응급 의료 정보 제공, 개인 맞춤형 응급처치 가이드, 의료 지원 연결 등을 포함합니다.
